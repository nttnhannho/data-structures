# data-structures
In this project, I will implement some popular data structures using Python programming language.
+ Built-in:
	- List
	- Tuple
	- Set
	- Dict
+ User-defined:
	- Array
	- Linked list
	- Stack
	- Queue
	- Tree
	- Hash Table
	- Graph
+ Collections:
	- Counter
	- Name Tuple
	- Order Dict
	- Default Dict
	- Deque
	- Frozen Set
+ Bonus data type:
	- String
